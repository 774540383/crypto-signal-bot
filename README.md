# 🤖 بوت الإشارات الاحترافي - Crypto Signal Bot

بوت تيليجرام احترافي لتحليل العملات الرقمية وتوليد إشارات التداول الآلية.

## ✨ المميزات

- 📊 **تحليل تقني متقدم**: EMA, RSI, ATR, Supertrend, BOS, Liquidity Sweep
- 🔄 **ماسح تلقائي**: فحص مستمر للأسواق واكتشاف الإشارات
- 💡 **تقييم الإشارات**: نظام نقاط شامل لتقييم جودة الإشارة
- 🎯 **إدارة رأس المال**: حساب حجم المركز والمخاطرة التلقائي
- 📈 **تتبع الأداء**: إحصائيات الربح والخسارة والمعدلات
- 🛑 **حماية من الخسائر**: إيقاف تلقائي بعد 3 خسائر متتالية

## 🚀 البدء السريع

### المتطلبات

- Python 3.11+
- حساب Binance API
- Token بوت Telegram
- معرف محادثة Telegram

### التثبيت

```bash
git clone https://github.com/774540383/crypto-signal-bot.git
cd crypto-signal-bot
pip install -r requirements.txt
```

### الإعدادات

قم بإنشاء ملف `.env`:

```bash
cp .env.example .env
```

ثم عدّل القيم:

```bash
BOT_TOKEN="your_telegram_bot_token"
CHAT_ID="your_chat_id"
SYMBOLS="BTCUSDT,ETHUSDT,SOLUSDT"
MIN_SCORE="70"
ACCOUNT_SIZE="100"
RISK_PCT="0.02"
CHECK_INTERVAL_SECONDS="300"
COOLDOWN_MINUTES="45"
```

### التشغيل

```bash
python bot.py
```

## 📱 أوامر البوت

| الأمر | الوصف |
|------|-------|
| `/start` | عرض معلومات البوت والأوامر المتاحة |
| `/signal` | البحث عن إشارة جديدة الآن |
| `/status` | عرض حالة البوت والإحصائيات |
| `/capital [رقم]` | تعيين رأس المال (مثال: /capital 100) |
| `/risk [1-3]` | تعيين نسبة المخاطرة (مثال: /risk 2) |
| `/win` | تسجيل ربح يدوي |
| `/loss` | تسجيل خسارة يدوية |

## ⚙️ الإعدادات المتقدمة

| المتغير | القيمة الافتراضية | الوصف |
|--------|------------------|-------|
| `BOT_TOKEN` | - | رمز بوت Telegram (مطلوب) |
| `CHAT_ID` | - | معرف المحادثة (مطلوب) |
| `SYMBOLS` | BTCUSDT,ETHUSDT,SOLUSDT | العملات المراد تحليلها |
| `TIMEFRAMES` | 15m,1h | الفترات الزمنية |
| `MIN_SCORE` | 70 | الحد الأدنى لقبول الإشارة |
| `ATR_MULT` | 1.5 | مضاعف ATR لحساب SL و TP |
| `RISK_PCT` | 0.02 | نسبة المخاطرة (2%) |
| `ACCOUNT_SIZE` | 100 | رأس المال الابتدائي ($) |
| `MAX_DAILY_SIGNALS` | 3 | الحد الأقصى للإشارات اليومية |
| `MAX_DAILY_LOSSES` | 3 | الحد الأقصى للخسائر المتتالية |
| `COOLDOWN_MINUTES` | 45 | فترة الانتظار بين الإشارات |
| `CHECK_INTERVAL_SECONDS` | 300 | فترة الفحص التلقائي |
| `PORT` | 10000 | منفذ الخادم |
